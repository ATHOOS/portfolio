Lors de notre dernière année de bachelier, nous avons effectué un voyage culturel à Malte.

Application MoviesAndMe

Afin de découvrir les bases de React Native, j'ai réalisé une petite application "MoviesAndMe"

  • Cette application est destinée à gérer une liste de films.
  • Elle utilise l'API TheMovieDataBase afin de fournir une liste de films à l'utilisateur en fonction de sa recherche.
  • L'utilisateur peut alors choisir un film, et soit l'ajouter à ses favoris, soit le marqué comme "lu".
  • L'application comprend également une liste de tous les films favoris que l'utilisateur a enregistré.

Le but de cette application était d'appliquer la matière vue en parallèle de la formation OpenClassrooms.

Raspberry Pi 4 - 6h Hébergement Site Web
Raspberry Pi - 6h

Hébergement Site Web

Afin de rendre mon portfolio accessible à tous, il m'a fallu trouver une solution pour l'héberger. J'ai choisi de le faire moi-même sur mon Raspberry Pi 4

  • J'ai commencé par installer Docker
  • J'ai installé des outils de sécurité (fail2ban, un firewall)
  • J'ai déployé un serveur Nginx
  • Et j'ai finalement déployé les fichiers via FTP sur mon Raspberry.
  • Pour finir, j'ai loué un service DNS chez OVH afin de pouvoir utiliser un nom de domaine et non plus une adresse IP pour accéder à mon site.

Le but de ce projet était de rester au maximum indépendant des services que certaines entreprises (comme OVH) peuvent fournir. Ayant un Raspberry à ma disposition le lien fut vite fait.
